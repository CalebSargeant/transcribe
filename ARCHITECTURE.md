## Security Considerations

1. **API Keys**: Stored in plaintext in `~/.transcribe/config.yaml`
   - Standard for CLI tools
   - File permissions: 644 (user readable)
   - Consider: chmod 600 for extra security

2. **Slack Webhook**: URL contains secret token
   - Don't commit config.yaml to git
   - Don't share webhook URL

3. **Video Privacy**: 
   - Videos processed locally
   - Only text sent to OpenAI
   - Original videos never uploaded

4. **Network Calls**:
   - OpenAI API (HTTPS)
   - Slack Webhook (HTTPS)
   - Both use TLS encryption
